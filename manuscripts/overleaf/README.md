# Overleaf Files

This folder is for Overleaf-ready templates that are safe to share. Do not
commit figures, tables, or LaTeX files containing restricted empirical results
unless they have been cleared for release.

For local drafting, use placeholders or synthetic examples. When final aggregate
figures are approved, the human analyst can stage them from the restricted HPC
workspace into a separate cleared release folder.

The LaTeX manuscript template should use Lato when possible:

```latex
\usepackage[default]{lato}
```
